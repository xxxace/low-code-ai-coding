# 2026-03-28 工作日志

## VoidContainer 高度自适应（第十六阶段 Step 4 支撑）

**完成时间**：2026-03-28 上午

**改动文件**：`prototype/src/renderer/VoidContainer.vue`（唯一改动文件）

**实现内容**：
- 新增 `voidWrapperRef` + 绑定到 `.void-wrapper` DOM
- 新增 `isAbsoluteMode` computed（`x-position-type === 'absolute'`）
- 改造 `wrapperStyle`：absolute 模式下从 nodeStyle 解构剔除 `height`，让内容自然撑开 wrapper
- 新增 `syncContainerSize()`：含双守卫（designMode+isAbsoluteMode 异步安全）+ 零尺寸守卫（Tabs 非激活页 display:none 返回全零时跳过）
- 新增 `onMounted` + `nextTick`：首次渲染同步（复用 FieldRenderer 模式）
- 新增 `ResizeObserver`（防抖 200ms）：监听 Tabs 切换 / Collapse 展开等动态内容变化
- 新增 `onBeforeUnmount`：清理 observer + 定时器，防止内存泄漏

**验证结果**：154 个单元测试全部通过，TypeScript 零错误，Vite 构建成功

**关键技术决策**：
- wrapperStyle 必须在 absolute 模式下剔除 height CSS，否则 wrapper 高度被锁死，ResizeObserver 空转
- 防抖选 200ms（非 150ms）：el-collapse 展开动画约 300ms，150ms 会读到中间态
- schema 的 `x-position.height` 从"CSS 控制值"变为"记录值"，由 syncContainerSize 持续写入
