# 2026-03-27 工作日志

## 联动规则体验重构（苏格拉底+第一性原理+奥卡姆剃刀）

### 问题分析
- **问题1**：Schema 中联动规则可以用 `$self.value` 自引用（无 dependencies），但界面"依赖"字段为空，信息对不上——用户困惑
- **问题2**：联动规则块在属性面板占用大量空间，后续属性会越来越多

### 解决方案
**第一性原理**：联动本质 = 表达式（when 字符串）+ 效果（fulfill/otherwise），不必区分 $self 与 $deps，直接编辑 when 字符串即可。
**奥卡姆剃刀**：属性面板只显示摘要（N 条规则 + 每条一行 when→fulfill），编辑全在弹窗。
