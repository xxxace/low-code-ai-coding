# 2026-04-12 工作日志

## Label/Toolbar 可读性优化（CanvasOverlay.vue）

修复了设计器中选中框 toolbar 的 3 类问题：

1. **边界遮挡修复**：新增 `getToolbarPlacement()` 函数，检测节点距 overlay 顶部空间，空间不足 36px 时翻转为 `inside` 模式（`top: 2px`），避免溢出被剪裁。同时 `.canvas-overlay` 加 `overflow: visible`。
2. **宽度自适应**：`action-buttons` 从 `display: flex` 改为 `display: inline-flex`，宽度收缩到内容宽度，不再与 selectedBox 同宽。
3. **Label 回退优化**：`getNodeLabel()` 新增 19 个组件名→中文映射，避免显示英文组件名。
4. **视觉细节**：加 box-shadow、font-weight: 500、max-width + ellipsis、white-space: nowrap、z-index: 20、border-radius 6px。
