## 一、表单组件

### 1.1 StdFormWrapper（标准表单包装器）

主表单容器，提供工具栏和统一布局。

```vue
<template>
  <StdFormWrapper :toolbar="true">
    <!-- 表单内容 -->
  </StdFormWrapper>
</template>
```

**Props:**
- `toolbar: boolean` - 是否显示工具栏

**工具栏按钮:**
```typescript
stdForm.toolbar.add = true      // 新增
stdForm.toolbar.edit = true     // 编辑
stdForm.toolbar.delete = true   // 删除
stdForm.toolbar.save = true     // 保存
stdForm.toolbar.rollback = true // 回滚
stdForm.toolbar.draft = true    // 草稿
stdForm.toolbar.submit = true   // 提交
stdForm.toolbar.audit = true    // 审核
stdForm.toolbar.disaudit = true // 反审核
stdForm.toolbar.print = true    // 打印
stdForm.toolbar.finish = true   // 完成
stdForm.toolbar.archive = true  // 归档
stdForm.toolbar.cancel = true   // 取消
```

### 1.2 FieldItem（字段项）

表单字段布局组件。

```vue
<FieldItem 
  label="编号" 
  :width="56" 
  required
  :disabled="false"
  independent
  mode="horizontal"
>
  <el-input v-model="form.PLNNO" />
</FieldItem>
```

**Props:**
| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| label | string | - | 字段标签 |
| width | number | auto | 标签宽度 |
| required | boolean | false | 是否必填（红色*标记） |
| disabled | boolean | false | 是否禁用 |
| independent | boolean | false | 是否独立（不受父级禁用影响） |
| mode | 'horizontal' \| 'vertical' | 'horizontal' | 布局模式 |
| bold | boolean | false | 标签加粗 |
| color | string | - | 标签颜色 |

**特性:**
- 自动继承 StdForm 的 disabled 状态
- 支持 i18n 标签翻译

### 1.3 FieldItemGroup（字段项组）

## 二、选择组件

### 2.1 RemoteSelect（远程下拉选择）

异步加载选项的下拉框。

```vue
<RemoteSelect
  v-model="form.TEAM"
  v-model:content-value="form.V_TEAMNAME"
  value-key="TEAMNO"
  label-key="TEAMNAME"
  :data-loader="getTeamList"
  :params="{ args: [form.CORPNO] }"
  default-first
  no-empty
  auto-load
  cache-id="team-list"
/>
```

**Props:**
| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| v-model | string \| number | - | 绑定值 |
| v-model:content-value | string | - | 显示文本 |
| valueKey | string | 'CODE' | 值字段 |
| labelKey | string | 'CNAME' | 显示字段 |
| dataLoader | Function | - | 数据加载函数 |
| params | Object \| Function | - | 加载参数 |
| defaultFirst | boolean | false | 默认选中第一项 |
| noEmpty | boolean | false | 不添加空选项 |
| autoLoad | boolean | false | 自动加载 |
| cacheId | string | - | 缓存标识 |
| disabled | boolean | false | 禁用 |

**Expose:**
```typescript
{
  getOptions: () => options.value,
  setOptions: (opts) => void,
  load: async () => void
}
```

**使用示例:**
```typescript
const getTeamList = async (params) => {
  return await fetchDataList<TeamVO[]>(
    request('队伍列表查询', params)
  )
}
```

### 2.3 DialogPicker（弹窗选择器）

带查询表格的弹窗选择器。

```vue
<DialogPicker
  v-model:visible="visible"
  title="客户查询"
  width="580px"
  height="600px"
  mode="confirm"
  :columns="columns"
  :manager="queryParamsManager"
  :loader="dataLoader"
  @confirm="handleConfirm"
>
  <template #search="{ onSearch, onReset }">
    <FieldItem label="编码">
      <el-input v-model="queryParams['A.CODE__like']" />
    </FieldItem>
    <el-button @click="onSearch">查询</el-button>
    <el-button @click="onReset">重置</el-button>
  </template>
</DialogPicker>
```

**Props:**
| 属性 | 类型 | 说明 |
|------|------|------|
| objectName | string | 国际化命名空间 |
| manager | RefManager | 查询参数管理器 |
| columns | Column[] | 表格列配置 |
| loader | Function | 数据加载函数 |
| multi | boolean | 是否多选 |
| autoLoad | boolean | 自动加载 |

**Expose:**
```typescript
{
  open: () => void,
  close: () => void,
  clear: () => void
}
```

## 三、表格组件

### 3.1 ReportTable（报表表格）

只读报表表格，支持汇总、键盘导航。

```vue
<ReportTable
  :columns="columns"
  :data="dataList"
  :footer="true"
  :readonly="true"
  @current-row-change="handleRowChange"
/>
```

**Props:**
| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| columns | Column[] | - | 列配置 |
| data | any[] | [] | 数据 |
| readonly | boolean | false | 只读 |
| footer | boolean | false | 显示汇总行 |
| footerData | any[] | - | 自定义汇总数据 |
| footerFormatter | Function | - | 汇总格式化 |
| checkMethod | Function | - | 复选框启用条件 |

**列配置:**
```typescript
const columns = [
  { type: 'seq', width: 60, align: 'center' },
  { field: 'PLNNO', title: '单号', width: 90, fixed: 'left' },
  { field: 'QTY', title: '数量', width: 80, align: 'right', summary: true },
  { field: 'BEGFR', title: '日期', width: 80, formatter: 'date' }
]
```

**Events:**
- `current-row-change(row)` - 当前行变更
- `cell-change(cell)` - 单元格变更
- `footer-row-updated(summary)` - 汇总更新

### 3.2 EditableTable（可编辑表格）

支持行编辑、添加删除、键盘操作的表格。

```vue
<EditableTable
  ref="tableRef"
  :columns="columns"
  :data="dataList"
  :tool-bar="true"
  :add-button="true"
  :remove-button="true"
  :footer="true"
  :row-initializer="rowInitializer"
  @add="handleAdd"
  @remove="handleRemove"
  @cell-value-change="handleCellChange"
/>
```

**Props:**
| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| columns | Column[] | - | 列配置 |
| data | any[] | - | 数据 |
| toolBar | boolean | true | 显示工具栏 |
| addButton | boolean | true | 显示添加按钮 |
| removeButton | boolean | true | 显示删除按钮 |
| footer | boolean | false | 显示汇总 |
| rowInitializer | Function | - | 行初始化函数 |
| beforeEditMethod | Function | - | 编辑前判断 |
| checkMethod | Function | - | 复选框启用条件 |
| removeMethod | Function | - | 删除前判断 |

**行初始化:**
```typescript
const rowInitializer = (row, utils) => {
  row.SEQ = utils.getMax('SEQ') + 1
  row.DT = utils.now().format('YYYY-MM-DD')
  return row
}
```

**Expose:**
```typescript
{
  getCurrentRow: () => currentRow,
  addRow: async (defaultData, targetRow) => row,
  remove: async (rows) => rows,
  removeRow: () => void,
  refreshCurrentRow: async () => void,
  updateRowField: async (row, field, value) => void,
  filter: async (params) => void,
  init: () => void
}
```

## 五、Hooks

### 5.1 useNsI18n（国际化）

```typescript
const { t, c } = useNsI18n('VUE_INQUIRY')

// t: 带命名空间的翻译
t('lbPLNNO', '编号')  // → 'VUE_INQUIRY.lbPLNNO'

// c: 通用翻译（不带命名空间）
c('common.search', '查询')
```

**支持默认值语法:**
```typescript
t('lbPLNNO|编号')  // 翻译失败时返回"编号"
```

### 5.3 useRefManager（响应式数据管理）

```typescript
// 对象类型
const [form, formManager] = useRefManager<InquiryOrderVO>(() => ({
  BLTY: 'IQYORD',
  STFG: 'BL_D'
}))

formManager.setRules(formRules)
formManager.update(data)  // 更新数据并保存old
formManager.reset()       // 重置为默认值
formManager.sync()        // old = value

// 数组类型
const [list, listManager] = useArrayRefManager<DetailVO>([])
listManager.setRef(tableRef)  // 关联表格组件
listManager.update(data)
listManager.records  // 获取表格数据

// 参数类型（自动初始化）
const [query, queryManager] = useParamsRefManager(() => ({
  PLNNO: '',
  BEGFR: ''
}))
```
